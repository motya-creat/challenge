## Инструкция
```
git checkout (название вашей ветки) - переход между ветками

git add .  (сохранить все файлы)
git commit -m 'сообщение (что вы сделали)'
git push (запушить изменения)

Запуск вашего сайта в вашем VSC в Terminal - 

* npm run dev

Если вылезает ошибка проверь установленную папку node_modules

Установка зависимосей (установит node_modules)

* npm install

```
